# Wave Team

A front-end portfolio and marketing site built with Vite, React, Tailwind CSS, and TanStack Start.

## Features

- Arabic is the default language mode.
- Dynamic UI translations for English and Arabic.
- Project cards render from `public/wave/script.js`.
- Contact form opens WhatsApp chat to `+20 15 50888640` with a prefilled message.

## Run locally

```bash
npm install
npm run dev
```

Then open the local Vite URL in your browser.

## Files of interest

- `public/wave/index.html` — main landing page structure.
- `public/wave/script.js` — translations, project rendering, language toggle, and WhatsApp form handling.
- `public/wave/styles.css` — custom styling for the Wave landing page.

## WhatsApp contact flow

Submitting the contact form opens a WhatsApp chat with a prefilled message to the phone number `+20 15 50888640`.
